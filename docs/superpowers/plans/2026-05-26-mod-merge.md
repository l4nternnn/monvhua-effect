# Mod 合并实现计划

> **面向 AI 代理的工作者：** 此计划涵盖将 clairvoyance-template-1.21.8 完整合并到 monvhua-effect。使用 superpowers:executing-plans 逐任务实现，复选框跟踪进度。

**目标：** 将 clairvoyance-template（千里眼/视线诱导/身体部件/镜像/搬运等）合并到 monvhua-effect，统一为 mod ID `monvhua`，Mojang Official Mappings。

**技术栈：** Java 21, Fabric Loom 1.16, Mojang Official Mappings, Fabric API 0.136.1+1.21.8
